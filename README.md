# Developer Portfolio (Frontend Only)

Modern portfolio project built with React.

## Tech Stack

- React
- React Router
- Tailwind CSS
- Framer Motion
- Vite

## Project Structure

```text
reactproject/
  client/
    src/
      components/
      data/
      pages/
```

## Data Handling

- Projects are hardcoded in `client/src/data/projects.js`.
- Contact form submissions are stored in frontend memory while the app is running.

## Run Locally

1. Install dependencies

```bash
cd client
npm install
```

2. Start development server

```bash
npm run dev
```

Runs at `http://localhost:5173` (default Vite port).
